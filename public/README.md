# sandcastle-frontend

## POST example

`/api/v1/your-game-name/lobby/34`

Example sent data: 
`{ username: "player1234", position: [123, -10, 200] }`
Received data: 
`{ players: [ { username: "player1234", position: [123, -10, 200] }, { username: "player9876", position: [987, 10, -200] } ] }`
